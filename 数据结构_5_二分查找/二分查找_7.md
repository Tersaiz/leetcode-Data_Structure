# 7. 寻找旋转排序数组中的最小值

>题目描述：假设按照升序排序的数组在预先未知的某个点上进行了旋转。
( 例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2] )。
请找出其中最小的元素。
你可以假设数组中不存在重复元素。

![示例](images\二分查找_7.png)

## 解题思路

+ 1.遍历保存最小值

## 代码（c++版本）

```c++
class Solution {
public:
    int findMin(vector<int>& nums) {
        int min_num = nums[0];
        int temp;
  
        
        for(int i=1;i<nums.size();i++)
        {
            temp = nums[i];
            if(temp < min_num)
            {
                min_num = temp; 
            }
        }
        return min_num;

    }
};
```
